# Sales Data Analysis (Past 5 Years)

## 📌 Project Description
This project analyzes sales data from the past five years to identify trends, patterns, and insights using Python.

## 🎯 Objectives
- Identify highest and lowest sales year
- Analyze yearly sales trends
- Visualize data using graphs

##System Architecture
Input Data (CSV)
↓
Data Cleaning (Pandas)
↓
Data Analysis (Grouping, Calculations)
↓
Visualization (Matplotlib)
↓
Insights / Output

## ⚙️ Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## 📊 Dataset
- Sample Superstore Dataset (CSV file)

## 🚀 How to Run
1. Open Jupyter Notebook
2. Run all cells in `Sale Data.ipynb`

## 📈 Output
- Year-wise sales analysis
- Graphs for visualization
